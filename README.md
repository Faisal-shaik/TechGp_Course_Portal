# 🎓 Online Course Portal

An interactive **Online Course Management System** built using **PHP**, **MySQL**, and **XAMPP**.  
This platform allows **Admins** to manage courses and **Students** to enroll, learn, and earn certificates online.

---

## 🚀 Features

### 👨‍🏫 Admin
- Add, edit, or delete courses  
- Manage enrolled students  
- Upload course content and certificates  
- View reports of course activity

### 👩‍🎓 Student
- Register and log in securely  
- Browse and search available courses  
- Enroll in courses and track progress  
- Download certificates after completion
